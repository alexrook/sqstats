Статистика Squid

Создано во VKEK IT Department под влиянием утилиты Sarg, исходный код доступен на GitHub

  • Данные загружаются с двух экземпляров Squid в PostgreSQL в соответствии с Squid default log format
  • Расчет итогов производится ежесуточно, по /etc/cron.daily, поэтому итоги для текущих суток не репрезентативны
  • Суммы байт учитывают, в том числе, обмен траффиком между Squid и клиентом
