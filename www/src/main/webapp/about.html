Статистика Squid

Создано VKEK IT Department под влиянием утилиты Sarg, исходный код доступен на GitHub

  • Данные загружаются с двух экземпляров Squid в PostgreSQL в соответствии с Squid default log format
  • Расчет итогов производится ежесуточно, еженедельно, ежемесячно по /etc/cron.(daily,weekly,monthly), поэтому итоги для текущих суток, недели, месяца не репрезентативны. Итоги за год вычисляются ежемесячно, поэтому верны для прошлых лет, а для текущего года - как накопительные на начало текущего месяца.
  • Суммы байт учитывают, в том числе, обмен траффиком между Squid и клиентом
  • Загрузки считаются для Content-Type's отмеченных как download. See maintenance->download content-types or contenttype relation.
